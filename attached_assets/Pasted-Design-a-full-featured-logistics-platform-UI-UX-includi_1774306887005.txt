Design a **full-featured logistics platform UI/UX** including:

1. Mobile App (iOS & Android) for individual users and drivers
2. Web Dashboard for corporate users
3. Admin Panel for system owner (super admin)

The platform is a hybrid of:

* Marketplace model (like load posting & bidding systems)
* Real-time vehicle dispatch (Uber-like matching)
* Live GPS vehicle tracking system

Language: **Turkish (ALL UI TEXT MUST BE IN TURKISH)**

---

# 🎯 DESIGN GOALS

* Mobile-first, fast and intuitive
* Clean SaaS + marketplace hybrid UI
* Designed for logistics, transportation and supply chain
* Minimal clicks, maximum automation
* Strong notification-driven experience (users should not need to search)

---

# 👥 USER ROLES (CRITICAL)

## 1. Super Admin (System Owner)

* Full control panel
* User management (individual + corporate)
* Listing control (approve/reject/feature)
* Revenue & performance analytics
* Advertisement management
* System monitoring

## 2. Corporate User (B2B कंपनies)

* Can post loads
* Can search loads
* Can receive and manage offers
* Has web dashboard
* Can create sub-users (staff accounts)
* Can access premium listings
* Can track shipments live

## 3. Individual User (Drivers / Small transporters)

* Mobile app only
* Can search loads
* Can post simple listings
* Can receive real-time job notifications
* Can accept jobs quickly

## Main Screens:

### 1. Home (Dashboard)

* Smart feed (no search needed)
* Nearby loads (auto-listed)
